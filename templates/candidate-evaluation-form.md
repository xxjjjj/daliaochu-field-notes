# 候选人评估记录表（脱敏骨架）

> 本模板对应 `playbooks/candidate-evaluation-framework.md` 的 §4.4，是其「可直接打印 / 抄写」的最小版本。
> 仅作为内部招聘 / ITBP / 团队负责人私下协作使用，**禁止公开、禁止外发、禁止写入任何对外系统**。

---

## 0. 基本信息（脱敏占位）

```text
候选人编号：____（不要写真名，建议按 YYYYMMDD-缩写编号，例如 20260601-HF）
评估日期：____
评估人：____
目标岗位方向：____（例：实施一组 / 数据组 / 内部 IT / 业务分析）
简历材料存放：____（仅本地路径，不上传）
```

> ⚠️ **以下信息一律不写入本表：**
> 真实姓名、身份证号、电话、邮箱、家庭住址、家庭成员、感情状态、
> 毕业院校全名、过往公司全名、过往薪资数字、宗教信仰、健康情况。

## 1. 维度一：值不值得培养

```text
锚点 1（学习自驱力）：
  - 行为事实：
  - 判断依据：

锚点 2（超出要求的小动作）：
  - 行为事实：
  - 判断依据：

锚点 3（不熟悉领域的反应方式）：
  - 行为事实：
  - 判断依据：

红牌信号（任一命中即高风险）：
  - □ 「精通 X」但讲不出 X 的核心取舍
  - □ 答不上来就沉默、不解释也不引导
  - □ 所有项目都是「完成作业」风格，没有自发探索痕迹
  - □ 其他：

红牌 / 绿牌综合判断：____（红 / 黄 / 绿）
```

## 2. 维度二：潜力在哪

```text
跨学科背景：____
小影响力痕迹：____
  - 例：组内分享、开源小项目、社群运营、自发 blog
现场学习速度：____（需现场观察，不能光听他说）
未来 3 年愿景清晰度：____（清晰 / 模糊 / 没想过）

绿牌信号（任一命中加分）：
  - □ 有从 0 到 1 的完整小闭环经验
  - □ 会主动记录方法 / 模板 / 笔记
  - □ 对自己不知道的事不装懂，主动问
  - □ 其他：

绿牌综合判断：____（强 / 中 / 弱）
```

## 3. 维度三：风险在哪

```text

  - 城市 / 通勤 / 经济压力：

文化冲突：
  - 沟通风格 / 反馈接收 / 团队偏好观察：

价值观：
  - 对加班 / 突发任务 / 失败的态度：

技能断层：
  - 简历写的能力 vs 实际深度（现场观察结论）：

期望错配：
  - 薪资 / 职级 / 成长速度预期（只记等级，不记数字）：
```

## 5. 综合输出

```text
星级评定：____（3 星匹配 / 2 星匹配 / 1 星匹配 / 不匹配）

一句话定性（不超过 30 字）：____

一段说明（不超过 100 字）：____

下一步建议（单选）：
  □ 建议推进（直接发 offer 或终面）
  □ 建议二面（不同面试官交叉验证）
  □ 建议储备（半年内合适岗位再激活）
  □ 建议放弃（明确不匹配，记录原因）
```
